# 📊 Student Marks Analyzer (NumPy)

## 📌 Overview

This project analyzes student marks using Python and NumPy.
It calculates average, highest, lowest marks and assigns grades.

---

## 🚀 Features

* Calculate average marks
* Find highest and lowest marks
* Assign grades (A, B, C, D)
* Simple console output

---

## 🛠 Tech Used

* Python
* NumPy

---

## ▶️ How to Run

```bash
pip install numpy
python main.py
```
